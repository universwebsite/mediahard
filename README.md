# MediaHard: Экспертный маркетплейс оборудования

**MediaHard** — это не просто интернет-магазин, а доверенная среда для купли-продажи железа с подтвержденным ресурсом.

## 💎 Бизнес-концепция
Проект решает проблему недоверия на рынке б/у оборудования. Каждая единица товара имеет **«Цифровой паспорт»**, что позволяет продавать экспертные решения на 15–30% выше среднерыночной цены.

## 🚀 Технологический стек и Инфраструктура
Проект спроектирован как высокодоступная система (High Availability):
*   **Виртуализация:** Кластер Proxmox (3 ноды).
*   **Хранилище:** Распределенная файловая система **Ceph** (обеспечивает сохранность данных «паспортов» даже при выходе из строя узла).
*   **Контейнеризация:** Docker-стек, развернутый поверх CephFS.
*   **Backend:** PHP 8.3 / Symfony (выбран за строгую архитектуру и надежность).
*   **Database:** PostgreSQL 15.
*   **AI-Ops:** Интеграция с локальной LLM (Ollama) для автоматизированного анализа логов стресс-тестов.

## 🛠 Ключевые инженерные решения
1.  **Digital Passport System:** «Единая точка правды» о товаре. Модуль хранит JSON-метаданные тестов, привязанные к серийному номеру, и медиа-доказательства (SMART, MemTest86). Модуль фиксации исправности (скриншоты тестов + видео упаковки).
2.  **Security Layer:** Безопасность уровня предприятия. Внедрена двухфакторная аутентификация (2FA) для админ-панели и неизменяемый лог аудита (Audit Trail) всех критических действий.
3.  **Incident Management:** Опыт эксплуатации кластера под нагрузкой (см. [docs/ARCHITECTURE.md](./docs/ARCHITECTURE.md)).
4.  **Monitoring:** Визуализация состояния железа и здоровья Ceph через Prometheus + Grafana.

## 🚀 CI/CD и Схема развертывания (IaC)
Реализован автоматизированный цикл поставки кода:
`GitHub (Push)` ➔ `Actions (Build & Test)` ➔ `Docker Hub` ➔ `VPS (Automated Deploy)`
Вся конфигурация инфраструктуры описана как код, что позволяет развернуть зеркало проекта за считанные минуты.
